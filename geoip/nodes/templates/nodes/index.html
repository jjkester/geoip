{% trans "Nodes" %}

The measurements performed by this application require a set of test nodes. A node is a physical computer system with a public IP address (eiter IPv4, IPv6 or both) and a known physical location. This location is used as a ground truth when looking up the node's IP address in IP geolocation databases. This way it is possible to calculate the accuracy of a database for a given node.

The measurements do not require access to the node, only the information about the node is sufficient. However, the information has to be accurate for the research method to work.

For more information about the research method see the about page.
